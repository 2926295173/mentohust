# Mentohust

锐捷校园网连接工具，支持OpenWRT/Ubuntu/Fedora.

# 开发说明
本项目为HustMoon开发的MentoHUST （<http://mentohust.googlecode.com/>）的接续。本软件采用The GNU Build System。
